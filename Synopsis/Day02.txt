JAVASCRIPT TRAINING SYNOPSIS – DAY 2
================================================================================

JavaScript concepts covered on Day 2,
focusing on modern syntax, array methods, and practical programming techniques.

--------------------------------------------------------------------------------
1. TEMPLATE LITERALS
--------------------------------------------------------------------------------
- Uses backticks (`) and ${} for dynamic string interpolation.

Example:
    let name = "Sindhu";
    console.log(`Hello ${name}`);

--------------------------------------------------------------------------------
2. ARROW FUNCTIONS
--------------------------------------------------------------------------------
- Short syntax for writing functions.

Example:
    let sum = (a, b) => a + b;
    console.log(sum(5, 10)); // 15

--------------------------------------------------------------------------------
3. ARRAY METHODS (FUNCTIONAL PROGRAMMING)
--------------------------------------------------------------------------------
- filter() → select elements
- map() → transform elements
- find() → locate element

Example:
    let arr = [1, 2, 3, "Sindhu"];
    let result = arr
        .filter(x => typeof x === "number")
        .map(x => x * 2);

--------------------------------------------------------------------------------
4. MATH FUNCTIONS
--------------------------------------------------------------------------------
- Used for generating random values and calculations.

Example:
    Math.floor(Math.random() * 100) + 1; // 1–100

--------------------------------------------------------------------------------
5. TYPE CONVERSION
--------------------------------------------------------------------------------
- Converting data between types.

Example:
    Number("5");      // 5
    String(5);        // "5"
    parseInt("10.5"); // 10

--------------------------------------------------------------------------------
6. ARRAY OPERATIONS
--------------------------------------------------------------------------------
- push() → add to end
- pop() → remove from end
- unshift() → add to beginning
- shift() → remove from beginning
- splice() → add/remove elements

Example:
    let colors = ["red", "green", "blue"];
    colors.push("yellow");
    colors.splice(1,1);

--------------------------------------------------------------------------------
7. REDUCE FUNCTION
--------------------------------------------------------------------------------
- Used to convert an array into a single value (sum, object, etc.)

Example (sum of array):
    const prices = [10, 20, 30, 40];

    const total = prices.reduce((acc, curr) => acc + curr, 0);

Explanation:
    acc  → accumulator (stores result)
    curr → current element
    0    → initial value

Dry Run:
    0 + 10 = 10
    10 + 20 = 30
    30 + 30 = 60
    60 + 40 = 100

Example (counting elements):
    const fruits = ['apple', 'banana', 'apple'];

    const count = fruits.reduce((acc, fruit) => {
        acc[fruit] = (acc[fruit] || 0) + 1;
        return acc;
    }, {});

Concepts covered:
    - Selecting elements using id and class

Navigation:
